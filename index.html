ESCAPE: A Standardized Benchmark for Multilabel Antimicrobial Peptide Classification

Sebastian Ojeda1, Rafael Velasquez1, Nicolás Aparicio1, Juanita Puentes1, Paula Cárdenas1, Nicolás Andrade1, Gabriel González1, Sergio Rincón1, Carolina Muñoz-Camargo1, and Pablo Arbeláez1
1Universidad de Los Andes, Colombia
Paper (PDF) arXiv Video Code Data

ESCAPE provides a unified dataset, tasks, and evaluation for multilabel AMP classification.

Abstract

ESCAPE introduces a standardized benchmark for multilabel antimicrobial peptide (AMP) classification. It unifies heterogeneous sources into consistent splits, label taxonomies, and metrics to enable rigorous comparison across models and training regimes. The benchmark includes tasks spanning AMP detection, activity-type multilabel classification (e.g., antibacterial, antifungal, antiviral, antiparasitic), and optional species-level target prediction. We release code, dataset definitions, and a public leaderboard to foster reproducible research in computational AMP discovery.

Benchmark Tasks & Evaluation

Core Tasks

  • AMP Detection (Binary): AMP vs. non-AMP.
  • Activity Type (Multilabel): Antibacterial, Antifungal, Antiviral, Antiparasitic.

Optional Tasks

  • Species-level Targets: Predict target organism/strain where available.
  • Robustness: Cross-dataset generalization & class-imbalance stress tests.

Metrics

We report mAP, macro-F1, AUC, and balanced accuracy per task. For multilabel tasks we include per-class and macro/micro summaries.

Dataset Card

Sources: ESCAPE harmonizes multiple curated AMP datasets into a single standardized format. Each source is cited in our paper and included with license-compliant access instructions.

Splits: Train/Validation/Test sets with stratification by activity types and data provenance where feasible.

Licensing: Redistribution follows original dataset terms. Certain datasets require external download.

Download: See the Code repository for scripts, checksums, and data loaders.
